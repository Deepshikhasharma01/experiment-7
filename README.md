# experiment-7
**Exp 2.3.1 – React + Express + Axios Integration:**
This experiment connects a React frontend with an Express backend using Axios to fetch data from a MongoDB database. It displays products in a responsive grid with loading indicators and error handling, demonstrating full-stack integration.

**Exp 2.3.2 – Redux Shopping Cart:**
This experiment implements state management using Redux Toolkit in a React application. It manages cart operations like add, remove, and update items, and persists data using localStorage for a better user experience.

**Exp 2.3.3 – Real-Time Chat Application (Socket.IO):**
This experiment develops a real-time chat app using Socket.IO with React and Express. It enables instant messaging, shows online users, typing indicators, and handles user connections dynamically.
